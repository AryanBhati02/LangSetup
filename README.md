# LangSetup — One-Click Language Installer for VS Code

> Install any programming language in one click — directly from a sidebar panel.
> No browser searches. No tutorials. No copy-pasting commands.

## Features
- 📦 Sidebar panel in the Activity Bar
- 🔍 Always fetches the LATEST version from official APIs
- ⬇ Downloads official setup files to your Downloads folder automatically
- 🧩 Installs all required VS Code extensions silently
- 📋 **Activity History** — tracks every install session with logs
- 🖥 Windows, macOS & Linux support
- 16 languages: C/C++, Java, Python, JavaScript, TypeScript, HTML/CSS, C#, PHP, Kotlin, Swift, Ruby, Rust, Go, Dart/Flutter, R, Lua

## Usage
1. Click the 📦 icon in the left Activity Bar
2. Select your OS
3. Pick languages
4. Click Install → Done ✅

## Development
```bash
npm install
npm run compile
# Press F5 to test
```
